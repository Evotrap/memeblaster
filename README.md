# 🎉 MemeBlaster

## Pour générer le .exe (toi uniquement, une fois)

1. Installe **Node.js** → https://nodejs.org (version LTS)
2. Double-clique sur **BUILD.bat**
3. Le fichier `dist/MemeBlaster Setup 1.0.0.exe` apparaît → partage-le à tes amis !

---

## Pour tes amis (ultra simple)

1. Télécharge `MemeBlaster Setup 1.1.0.exe`
2. Double-clique → installe
3. L'app s'ouvre → colle le **Token du bot** + l'**ID du channel**
4. Clique **Connecter** → c'est bon !

L'app se lance toute seule au démarrage de Windows et se cache dans les icônes système.

---

## Créer le bot Discord (une seule fois pour tout le groupe)

1. https://discord.com/developers/applications → **New Application**
2. Onglet **Bot** → **Add Bot** → copie le **Token**
3. Active **Message Content Intent**
4. Invite le bot sur votre serveur (OAuth2 → bot → Read Messages)
5. Dans Discord : Paramètres → Avancés → **Mode développeur**
6. Clic droit sur le channel → **Copier l'identifiant**

→ Partage le token + l'ID du channel à tout le groupe

---

## Utilisation

Envoie dans le channel Discord : images, GIFs, vidéos, texte  
→ Tout le monde voit la popup sur son écran 🎉

Icône dans la barre système :
- Double-clic → Paramètres
- Clic droit → Activer/Désactiver ou Quitter
